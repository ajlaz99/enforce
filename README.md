## 🛠 How it works
- Static site (HTML, CSS, JS)
- Memes stored in `memes.js`
- Media in `/images`

## ✏️ How to add a meme
1. Upload image/video to `/images`
2. Add a new object in `memes.js`:
{
title: "...",
image: "images/your-file.jpg",
type: "image",
areas: ["events"]
}
